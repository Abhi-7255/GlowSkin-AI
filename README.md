# 🌟 GlowSkin AI

An AI-powered Skin Type Detection Web Application built using Flask, TensorFlow, Deep Learning, HTML, CSS and JavaScript.

## 🚀 Features

- AI Skin Type Detection
- Upload Image Analysis
- Live Selfie Capture
- Face Detection using OpenCV
- Personalized Skin Care Recommendations
- Contact Form with Email Notifications
- Fully Responsive UI
- Modern SaaS Inspired Design

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Flask
- Python

### AI / ML
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pillow
